# Miracle Collections

Handcrafted men's, women's, and kids' wear plus sandals and shoes, made with natural materials and timeless design.

## Features

- **Apparel**: Unique collections of handcrafted clothing.
- **Footwear**: Custom and handcrafted sandals and shoes.
- **High Quality**: Using premium materials.

## Development

This project is built using:

- **Framework**: Vite + React
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui

### Getting Started

1. **Install Dependencies**:
   ```sh
   npm install
   ```

2. **Start the Development Server**:
   ```sh
   npm run dev
   ```

3. **Build for Production**:
   ```sh
   npm run build
   ```
